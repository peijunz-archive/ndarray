# N-Dimensional Array class template
A simple `ndarray` class template using C++11.
## Aim
In C/C++, it is hard to pass a two dimensional array. This provides a simple
multidimensional array data structure for basic use. It has used Variadic
Template in C++11 to improve performance in indexing by `operator()`. The doc
can be generated by doxygen. In the `ndarray` class template, only datatype
and dimension are template parameter. The width of a specific axis is specified
at run time.

## Implemented features
+ Use `operator[]` for raw index
+ Use `operator()` for multidimensional index
+ Big five
+ Elementwise `+ - * /`
+ Iterator
+ Index rolling
